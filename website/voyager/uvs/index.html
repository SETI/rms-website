--- layout: base layout_style: default tabs: voyager_uvs --- ## Overview Voyager's Ultraviolet Spectrometer (UVS) instrument acquired profiles of the ring systems of Saturn, Uranus, and Neptune. This data set has been archived by the Ring-Moon Systems Node as volume [VG_2802](/webapps/link/volumes/VG_28xx/VG_2802/). ## The UVS Tabs The tabs associated with this page describe the UVS data set and how to obtain ring occultation profiles. * [Instrument](instrument.html) \- Descriptive information about the instrument. * [Archive Data](data.html) \- Information about and links to the UVS occultation data set published by the PDS Ring-Moon Systems Node. * [Standard Profiles](profiles.html) \- Links to ASCII tables of ring opacity vs. radius.
