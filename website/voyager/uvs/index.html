--- layout: base layout_style: default title: Voyager Ultraviolet Spectrometer (UVS) tabs: voyager_uvs --- # {{ page.title }} ## Overview Voyager's Ultraviolet Spectrometer (UVS) instrument acquired profiles of the ring systems of Saturn, Uranus, and Neptune. This data set has been archived by the Ring-Moon Systems Node as volume [VG_2802]({{ site.viewmaster_url }}volumes/VG_28xx/VG_2802/){:target="_blank"}. ## The UVS Tabs The tabs associated with this page describe the UVS data set and how to obtain ring occultation profiles. * [Instrument](instrument.html) \- Descriptive information about the instrument. * [Archive Data](data.html) \- Information about and links to the UVS occultation data set published by the PDS Ring-Moon Systems Node. * [Standard Profiles](profiles.html) \- Links to ASCII tables of ring opacity vs. radius.
