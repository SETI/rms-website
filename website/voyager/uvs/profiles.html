The columns tabulated are: * Radius in km * Raw PPS counts * Standard deviation in raw PPS counts * Normal optical depth * One standard deviation lower limit on optical depth * One standard deviation upper limit on optical depth (**Note**: Due to lack of calibration information, only the first three columns are tabulated for the second and third ring occultation profiles from Saturn---delta Sco ingress and iota Her egress.) Click on the data file name (*.TAB) to retrieve the data. Click on the corresponding label file (*.LBL){:target="_blank"} to retrieve the PDS label, which describes the data file contents in detail. Note that all of these files are located within subdirectories of the [EASYDATA]({{ site.viewmaster_url }}volumes/VG_28xx/VG_2802/EASYDATA/){:target="_blank"} directory on PDS volume [VG_2802]({{
