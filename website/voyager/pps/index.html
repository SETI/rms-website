--- layout: base layout_style: default tabs: voyager_pps --- ## Overview Voyager's Photopolarimeter Subsystem (PPS) instrument acquired fine-resolution stellar occultation profiles of the ring systems of Saturn, Uranus, and Neptune. This data set has been archived by the Ring-Moon Systems Node as volume [VG_2801]({{ site.viewmaster_url }}volumes/VG_28xx/VG_2801/){:target="_blank"}. ## The PPS Tabs The tabs associated with this page describe the PPS data set and how to obtain ring occultation profiles. * [Instrument](instrument.html) \- Descriptive information about the instrument. * [Archive Data](data.html) \- Information about and links to the PPS data set as published by the PDS Ring-Moon Systems Node. * [Standard Profiles](profiles.html) \- Links to ASCII tables of ring opacity vs. radius.
