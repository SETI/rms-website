Dear Mitch,

 

I have completed my review of the Voyager volumes.

 

On every volume, there was no CATINFO.TXT file in the DOCUMENT

sub-directory.  This is required.

 

Fixed.

 

On every volume there was no cumulative index.  This is required.

 

Fixed.

 

As far as the format of the files. I have attached a zipped tar file

with what I found on each volume.  The file names on the tar file are

of the form testlinVgIssxx, where the xx inidcates the volume number

for which the comments are intended.

 

I can summarize the errors as follows:

 

(1) DOCUMENT/TUTORIAL.TXT has some 82-character lines (including the <CR><LF>).

 

These only occur where we are quoting from the VICAR labels. We prefer to quote this text exactly, which means the line length cannot be shortened.

 

(2) Volume VGISS_0030 had some extraneous files:

S_RINGS/C3517XXX/._C3517118_RAW.LBL

S_RINGS/C3517XXX/._C3517118_GEOMA.LBL

 

These were artifacts of the Mac OS and have been removed.

 

(2) Volume VGISS_0038 had long list of errors.

 

These appear to have been errors that cropped in when the hard drive was written. They were never present in our copy. We regret the problems caused by unreliable disk drives.

 

 

 

Let me know if you have any questions.

 

Sincerely,
