VG_2802 Peer Review Summary

Voyager UVS Ring Occultation Data Set
Mark Showalter
September 30, 2004

Given the panel's consensus, this data set is now officially released. The only changes from the "near_final" version of the data set were a few syntax and editorial corrections.

The final version is now on line as a browsable directory VG_2802 and as a single, downloadable zip file VG_2802.zip. The previous versions have been removed. Thanks to one and all!

--Mark

September 15, 2004

This data set is about to be delivered to the Central Node for final ingestion into the PDS. Thank you very much to the hardy souls who submitted peer reviews.

The near-final version is on line as an expanded directory tree VG_2802_near_final or as a single, downloadable zip file VG_2802_near_final.zip.

It would be conventional to hold a telecon to discuss the data set and all the reviewer comments. However, in this case I have adopted nearly every recommendation submitted by the review panel. In those few cases where I did not do exactly what was recommended, I made alternative changes that I think address the issue adequately. See the detailed responses below.

I am hoping we can achieve a consensus that the data set is final and avoid the (always difficult to schedule!) step of holding a telecon.

Please let me know if you find this acceptable.

--Mark
