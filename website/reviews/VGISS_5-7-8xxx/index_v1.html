Mark Showalter and Mitch Gordon
May 21, 2013

Background

These three data sets contain calibrated and geometrically corrected versions of the Voyager images originally submitted to PDS. We previously prepared the Saturn encounter data which has now completed peer review and lien resolution. These three new data sets are constructed to parallel the Saturn data.

  1. We generate a complete set of dark currents for each encounter based on our experience with the Saturn data.
  2. While the original volumes were organized in directories based on the intended target, we have reorganized the new volumes into chronological order. The new names are:
    • Jupiter: VGISS_5101-5120 for Voyager 1, VGISS_5201-5214 for Voyager 2.
    • Uranus: VGISS_7201-7207 for Voyager 2.
    • Neptune: VGISS_8201-8210 for Voyager 2.
    The volumes break at fairly natural breakpoints in FDS count, and the sizes have been kept to less than approximately 8 GB per volume, following PDS recommendations.
  3. We have provided a well-stretched, full-resolution JPEG of every image. These can be found in the accompanying BROWSE directories.
  4. We have provided a lot of information in the documentation files.
  5. We have identified and corrected a number of errors in image metadata.
A few things are not part of this release:
  1. We have not yet augmented the geometric information about the images. This is the subject of ongoing work at Rings Node, along with similar work for the Cassini mission. This work is incomplete at this time but will be included in a future update to the data set.
  2. We have not yet corrected targets currently identified as "UNK_SAT". These will be corrected along with the augmented geometry information.
  3. We have not yet documented a few very old file formats that are archived primarily to support current users of VICAR. Charlie Avis has agreed to provide some of this information, and we will include it in an update.
