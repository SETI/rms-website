Cassini ISS and VIMS Migration Peer Reviews

Mike Evans and Mitch Gordon
December 13, 2019

Background

These data sets were previously archived by the PDS Imaging Node under PDS3. This project was migration to PDS4, performed by the PDS Ring-Moon Systems Node. These four PDS4 bundles contain the raw data sets for the Cassini Imaging Science Subsystem (ISS) and the Cassini Visual and Infrared Mapping Spectrometer (VIMS).

  • The bundle cassini_iss_cruise contains ISS data from 1999 to 2003
  • The bundle cassini_iss_saturn contains ISS data from 2004 to 2017
  • The bundle cassini_vims_cruise contains VIMS data from 1999 to 2003
  • The bundle cassini_vims_saturn contains VIMS data from 2004 to 2017

The reviews were conducted largely by email. Responses from the reviewers are below.
