--- layout: base layout_style: default title: "CIRS Documentation" --- ## 1.Documentation Please read the documentation carefully. The CIRS team has made a remarkable effort to provide useful information, including providing calibrated spectra for all of their data. * [About Original Format Data](orig_about.html) * [Naming Conventions](orig_dir_names.html) * [Finding Data](orig_finding.html) **Users are strongly encouraged to review a COCIRS_xxxx/COCIRS_xxxx/AAREADME.TXT file.** This file appears in the root directory of every volume and provides a good overview of the data set. A number of additional documents are included in the DOCUMENT subdirectory on each volume. Those files are summarized in the DOCINFO.TXT file. A copy of this file is included in the DOCUMENT subdirectory of every original volume. In particular, further details of the CIRS archive volumes can be found in the "Software Interface Specification" (SIS) Documents, found in the DOCUMENTS directory on each volume. * The Data Product and Label SIS are available as DATASIS.PDF and DATASIS.TEX. * The Volume Organization SIS are available as VOLSIS.PDF and VOLSIS.TEX. The CATALOG subdirectory on each volume contains a wealth of information about the CIRS instrument, team and data set. In particular: * DATASET.CAT: Description of the data sets. * INST.CAT: Description of the CIRS instrument. * INSTHOST.CAT: Description of the Cassini spacecraft. * MISSION.CAT: Description of the Cassini mission. The file, ERRATA.TXT, in the root directory of each volume, documents any known anomalies or deviations from PDS standards contained in that volume. Additional descriptions of the instrument can be found at the [Cassini CIRS web page](//saturn.jpl.nasa.gov/composite-infrared-spectrometer/){:target="_blank"} maintained at JPL. ## 2\.
